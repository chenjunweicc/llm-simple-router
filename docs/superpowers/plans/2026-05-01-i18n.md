# i18n 国际化实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为 llm-simple-router 管理后台实现全面的中英文国际化，覆盖所有页面、业务组件和工具函数。

**Architecture:** 使用 vue-i18n@11 + @intlify/unplugin-vue-i18n Vite 插件，Composition API 模式。翻译文件按语言目录（zh-CN/en）+ 模块拆分。语言切换放在 Sidebar 底部。后端 API 消息保持中文，前端通过 `t()` 翻译，后端中文作为 fallback。

**Tech Stack:** vue-i18n@11, @intlify/unplugin-vue-i18n, date-fns locale, Intl.NumberFormat

**Design doc:** `docs/plans/2026-05-01-i18n-design.md`

- [ ] **Step 1: 创建 `frontend/src/i18n/index.ts`**

```ts
import { createI18n } from 'vue-i18n'

const SUPPORTED_LOCALES = ['zh-CN', 'en'] as const
export type SupportedLocale = typeof SUPPORTED_LOCALES[number]

const STORAGE_KEY = 'llm-router-locale'
const DEFAULT_LOCALE: SupportedLocale = 'zh-CN'

function getInitialLocale(): SupportedLocale {
  const stored = localStorage.getItem(STORAGE_KEY)
  if (stored && SUPPORTED_LOCALES.includes(stored as SupportedLocale)) {
    return stored as SupportedLocale
  }
  return DEFAULT_LOCALE
}

export const i18n = createI18n({
  legacy: false,
  locale: getInitialLocale(),
  fallbackLocale: 'zh-CN',
  messages: {},
})

/**
 * 通过 import.meta.glob 动态加载指定语言的全部模块翻译。
 * 首次加载时在 main.ts 中调用，切换语言时在 useLocale 中调用。
 */
export async function loadLocaleMessages(locale: SupportedLocale): Promise<void> {
  const modules = import.meta.glob<{ default: Record<string, unknown> }>('./locales/*/*.json')
  const localeDir = `./locales/${locale}/`

  const loadJobs = Object.entries(modules)
    .filter(([path]) => path.startsWith(localeDir))
    .map(async ([path, loader]) => {
      const mod = await loader()
      const fileName = path.split('/').pop()!.replace('.json', '')
      return { fileName, messages: mod.default }
    })

  const results = await Promise.all(loadJobs)
  for (const { fileName, messages } of results) {
    i18n.global.mergeLocaleMessage(locale, { [fileName]: messages })
  }
}

/** 获取当前 locale（类型安全） */
export function getCurrentLocale(): SupportedLocale {
  return i18n.global.locale.value as SupportedLocale
}

export { STORAGE_KEY, SUPPORTED_LOCALES, DEFAULT_LOCALE }
```

import { computed } from 'vue'
import { zhCN, enUS } from 'date-fns/locale'
import type { Locale } from 'date-fns'
import { i18n } from './index'

const LOCALE_MAP: Record<string, Locale> = {
  'zh-CN': zhCN,
  en: enUS,
}

/** 获取当前语言对应的 date-fns locale（响应式） */
export function useDateLocale() {
  const dateLocale = computed(() => LOCALE_MAP[i18n.global.locale.value] ?? zhCN)
  return { dateLocale }
}

/** 格式化数字（按 locale 添加千分位等） */
export function formatNumber(value: number, locale: string): string {
  return new Intl.NumberFormat(locale).format(value)
}

/** 格式化百分比（0-1 范围） */
export function formatPercent(value: number, locale: string, fractionDigits = 1): string {
  return new Intl.NumberFormat(locale, {
    style: 'percent',
    minimumFractionDigits: fractionDigits,
    maximumFractionDigits: fractionDigits,
  }).format(value)
}

- [ ] **Step 4: 创建 `frontend/src/composables/useLocale.ts`**

```ts
import { computed } from 'vue'
import { useI18n } from 'vue-i18n'
import { loadLocaleMessages, STORAGE_KEY, type SupportedLocale } from '@/i18n'

/**
 * 语言切换 composable。
 * - locale: 当前 locale（响应式）
 * - setLocale: 切换语言并持久化 + 动态加载翻译
 * - localeLabel: 当前语言的显示名（用于切换按钮）
 */
export function useLocale() {
  const { locale } = useI18n()

  async function setLocale(lang: SupportedLocale) {
    locale.value = lang
    localStorage.setItem(STORAGE_KEY, lang)
    document.documentElement.setAttribute('lang', lang)
    await loadLocaleMessages(lang)
  }

  const localeLabel = computed(() => {
    return locale.value === 'zh-CN' ? '中文' : 'EN'
  })

  const toggleTarget = computed<SupportedLocale>(() => {
    return locale.value === 'zh-CN' ? 'en' : 'zh-CN'
  })

  return { locale, setLocale, localeLabel, toggleTarget }
}
```

- [ ] **Step 1: 改造 utils/format.ts**

当前 `formatTime` 和 `formatTimeShort` 硬编码了 `'zh-CN'` locale。需要让它们接受 locale 参数：

```ts
export function formatTime(iso: string, locale = 'zh-CN'): string {
  return parseUtc(iso).toLocaleString(locale, TZ_OPTS)
}

export function formatTimeShort(iso: string, locale = 'zh-CN'): string {
  return parseUtc(iso).toLocaleString(locale, {
    ...TZ_OPTS,
    month: '2-digit',
    day: '2-digit',
    hour: '2-digit',
    minute: '2-digit',
  })
}

export function formatTimeHM(date: Date, locale = 'zh-CN'): string {
  return date.toLocaleTimeString(locale, { ...TZ_OPTS, hour: '2-digit', minute: '2-digit' })
}
```

**注意：** 所有调用 `formatTime` 的组件需要同步更新，传入 `locale` 参数。可以在每个组件中从 `useI18n()` 获取 locale，或使用 `useDateLocale()` composable。

实际上为了减少改动量，更好的方案是直接读取 i18n 的当前 locale：

```ts
import { i18n } from '@/i18n'

function currentLocale(): string {
  return i18n.global.locale.value
}

export function formatTime(iso: string): string {
  return parseUtc(iso).toLocaleString(currentLocale(), TZ_OPTS)
}
```

这样调用方不需要改。

# 排除 i18n 目录下的文件
if '/i18n/' not in relative_path and 'i18n/' not in relative_path:
    in_template = False
    for i, line in enumerate(lines, 1):
        if '<template' in line and '</template>' not in line:
            in_template = True
            continue
        if '</template>' in line:
            in_template = False
            continue
        if not in_template:
            continue

        stripped = line.strip()
        if not stripped or stripped.startswith('<!--'):
            continue
        # 跳过已经使用 t() 或 $t() 的行
        if 't(\'' in line or 't("' in line or '$t(' in line:
            continue
        # 跳过组件属性中的动态绑定
        if ':placeholder' in line or ':title' in line or ':label' in line:

        # 检查是否包含中文字符
        for char in stripped:
            cp = ord(char)
            if 0x4E00 <= cp <= 0x9FFF:
                issues.append(f"  [第{i}行] 禁止在模板中硬编码中文文本")
                issues.append("    请使用 t() 函数进行国际化，如 {{ t('module.key') }}")
