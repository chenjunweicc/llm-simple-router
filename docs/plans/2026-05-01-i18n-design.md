# i18n 国际化设计方案

## 决策记录

| 决策项 | 选择 |
|--------|------|
| 目标语言 | 中文 (zh-CN) + 英文 (en) |
| 后端处理 | 前端 i18n，后端保持中文 |
| i18n 库 | vue-i18n@11 + @intlify/unplugin-vue-i18n |
| 翻译文件结构 | 按语言目录 + 嵌套模块 |
| 覆盖范围 | 全面覆盖（所有 views + 业务组件 + 工具函数） |
| 语言切换 | Sidebar 底部按钮 |
| 日期/数字格式化 | date-fns locale + Intl.NumberFormat |
| Key 命名风格 | 路径式 (e.g., dashboard.stats.totalRequests) |

## 改造模式

### 模板中
```vue
<!-- 前 --><h2>仪表盘</h2>
<!-- 后 --><h2>{{ t('dashboard.title') }}</h2>
```

### Script 中
```ts
// 前
toast.error(getApiMessage(e, '检查失败'))
// 后
const { t } = useI18n()
toast.error(getApiMessage(e, t('sidebar.checkFailed')))
```

### 动态文本
```vue
<!-- 前 -->
<p>{{ isUpgrading ? '升级中...' : '一键升级' }}</p>
<!-- 后 -->
<p>{{ isUpgrading ? t('sidebar.upgrading') : t('sidebar.oneClickUpgrade') }}</p>
```

## 后续维护规范

- 新组件/页面所有文本必须用 `t()`
- git hook 新增规则检测 Vue 模板中的硬编码中文
- zh-CN/*.json 和 en/*.json 的 key 必须一一对应
